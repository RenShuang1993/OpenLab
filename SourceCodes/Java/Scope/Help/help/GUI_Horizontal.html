Horizontal Section

The following image below illustrates the 'Horizontal Section'.

The horizontal knob is utilized to adjust the factor 'Time per Division'. This scaling factor has a close connection between the sampling rate. Thus, whenever the factor 'Time per Division' changes the sampling rate is adjusted accordingly. The horizontal knob has a value area from 1μs - 1s. The GUI does not support any possibility to set the sample rate manually!

Hint: If the horizontal knob is set to lower than 10ms/Div, the signal acquisition takes longer and the signal graph is not updated immediately. However, the application has not crashed!

Every OpenLab device supports the ETS (Equivalent Time Sampling) mode. This mode is entered whenever the maximum sample rate of the corresponding device is exceeded. The ETS lamp turns red whenever the device is in the ETS mode. Within the ETS mode only periodically signal can be analysed and measured. For further details about the ETS mode, you can click the label 'ETS:' or click on the following link: Equivalent Time Sampling
