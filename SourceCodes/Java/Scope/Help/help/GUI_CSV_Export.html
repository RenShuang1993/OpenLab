CSV Export

CSV Export

The GUI provides the functionality to export the signal graph to a .csv file. It is recommended to configure the CSV Export settings before the export is done, otherwise the default settings are applied and the result might not be satisfied. The settings can be adjusted in the menu bar under the section Settings->CSV Export. The 'CSV Export Settings' dialog is shown in the image below. Additionally to the entered file name, an index and a timestamp is added per default.

Clicking 'Export to .csv' (menu bar: File->Export to .csv) or the shortcut Ctrl + E sets the signal graph in the Stop state and exports the signal graph.
