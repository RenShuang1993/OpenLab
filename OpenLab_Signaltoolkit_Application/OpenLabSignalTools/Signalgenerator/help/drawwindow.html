    Draw Waveform Window

    Find it under View -> Draw Wave

    This windows allows you to graphically specify a waveform. It works by selecting points and then connecting them with interpolation.

    Linear Interpolation

    Connects the points with straight lines

    Cosine Interpolation

    The interconnection is based on the cosine function

    Cubic Interpolation

    The interconnection is based on a 3rd order polynomial. The resulting curve is slightly tighter than the curve produces by the cosine interpolation.

    Catmull-Rom Splines

    Also based on 3rd order polynomials, but additionally uses the first derivative for interpolation. The segments tend to be connected more continuous. The curves are not as tight compared to the cosine and cubic interpolation and might overshoot.

    Usage

    Ctrl+Leftclick: Creates a new "waypoint" at the cursors position
    Rightclick: Deletes the "waypoint" under the cursor

    Points can be dragged with the left mouse button. The outermost points cannot be removed nor moved horizontally as there are necessary for interpolation.
